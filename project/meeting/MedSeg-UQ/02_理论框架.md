## 0. 一句话主张（thesis）

在分割任务中，只要数据存在 **aleatoric 歧义**（即存在像素的真实条件前景概率 p(x)∈(0,1)，典型为器官/病灶**边界**），那么「分割精度（Dice）高」与「概率校准好（calibration error 小）」在数学上**不可兼得**：我们给出第一个形如

> **𝔇(f) ≥ α ⟹ ECE₁(f) ≥ Φ(α) = (1/3)(α·E[p] − E[p²])** 的形式化下界（定理 P2，**致命洞已修、用实用 ECE 度量**），

其中 Φ>0 ⟺ α 超过校准最优 𝔇_cal\*=E[p²]/E[p]，且**可买头空间 = E[p(1−p)]/E[p] = 归一化 aleatoric 不确定性**——数据无歧义（p∈{0,1} a.e.）时退化为 0，精确刻画「trade-off 由谁驱动、何时消失」。这解释了 mL1-ACE 等工作只能实证观察、却无法解释的现象（[2506.03942] 自称 "inherent tension" 但无定理）。
> **诚实边界**：当前结果在建模假设 A1（large-image 极限）下成立，且 P1 在 A1 下是已知 calibration-refinement 分解的推论——**真正的顶会贡献在去 A1（有限-N U-statistic）+ 升级为 minimax 信息论下界**，路线见 §8。

## 2. 形式化设定（setup）

**逐像素概率模型**。图像域像素集 𝒳（有限），对每像素 x：
- 真实条件前景概率 **p(x) = P(y=1 | x) ∈ [0,1]**（数据分布固有，含 aleatoric 歧义）。
- 模型输出 **f(x) ∈ [0,1]**（soft 前景概率）。
- 硬预测 ŷ(x) = 𝟙[f(x) > τ]，真值 y(x) ∈ {0,1}。
- 像素按某分布 𝒟 抽样（或固定图像上的经验分布）。

**歧义量（ambiguity mass）**。定义 δ-歧义集与其质量：
> 𝓐_δ = { x : p(x) ∈ [δ, 1−δ] }，  m_δ = P_𝒟(x ∈ 𝓐_δ)。
m_δ 度量「真值本身就模糊」的像素占比（边界区）。**主张：trade-off 的强度由 m_δ 驱动。**

**校准误差度量**。采用 Błasiok 的 smooth calibration error（比 ECE well-behaved，且与 ℓ1-distance 多项式等价）：
> **smCE(f) = sup_{g: 1-Lipschitz, [0,1]→[−1,1]} E_𝒟[ g(f(x)) · (y(x) − f(x)) ]**。  `【已证积木·待核原文】`形态见 [2211.16886]，精确常数待核。

**分割精度度量**。期望 soft-Dice 或阈值化 Dice：
> D(f) = E[ 2·|f∩y| / (|f|+|y|) ]（soft），或硬版按 ŷ 算。先用「逐像素 overlap 期望」的可分解代理简化，再推广（见 §5 难点）。

## 3. 证明骨架（three-step）

### Lemma 1 —— Dice 偏好硬决策（sharpening）
> **直觉**：Dice 是 overlap 奖励，期望 Dice 的最优解是**ranking top-τ* 选择规则（硬 0/1）**，不是输出真实概率 p(x)。故在歧义区 𝓐_δ，Dice-最优预测必然把 f 推向极端。

- 积木（**已从主源核实**，[2206.13086] RankSEG）：
  - **Thm 1（Dice-Bayes rule）**`【已核】`：最优规则 δⱼ*(x) = 𝟙[ pⱼ(x) 排进 top-τ*(x) ]，否则 0。即**按像素概率排序、选前 τ* 个**的 top-k 型规则，最优体积 τ*(x) 由最大化 Poisson-binomial 表达式（其 Eq.4）决定，**数据自适应**。
  - **Remark 2**`【已核】`：「Dice-Bayes rule 不太可能由固定阈值框架得到，因最优阈值随输入剧烈变化」→ 固定阈值对 Dice **次优**。
  - **Def 8 + Lemma 9**`【已核】`：定义 Dice-calibration；证 CE / focal 等分类 loss **非 Dice-calibrated**；soft-Dice 只近似不保证最优。
- 含义（精化）：Dice-最优是**硬选择**（top-τ*），在 𝓐_δ 上必然把被选中像素推向 f→1、未选推向 f→0，与校准目标 f→p(x)∈(δ,1−δ) 直接冲突。**且 RankSEG 已证连分类 loss 都非 Dice-calibrated → 想要 Dice 最优就得离开「输出真实概率」的校准解**——这正面支撑 trade-off 存在性。
- `【待证·我方】`**量化版 Lemma 1**：仍需把「Dice(f) ≥ α」（approximate，非 exact Bayes）翻译成「f 在 𝓐_δ 上偏离 p 至少 s(α)」的**定量 sharpening 下界**。RankSEG 给的是 exact 最优的结构（top-τ* + Remark2 + Lemma9 定性冲突），approximate 版的定量 s(α) 是我方核心增量。

### Lemma 2 —— 歧义区的 sharp 预测必然 miscalibrated
> **直觉**：若像素 x 真实 p(x)∈(δ,1−δ) 却被预测成 f(x)≈0 或 1，则 f(x) 系统性偏离真实频率 → 贡献 calibration error。

- `【待证·我方】`用 smCE 定义**直接构造** 1-Lipschitz 测试函数 g*，使
  > smCE(f) ≥ E_𝒟[ g*(f(x))·(y−f(x)) ] ≥ (歧义区上 |f−p| 的加权积分) − 余项。
  关键：在 𝓐_δ 上 sharp（|f−p|≥s）的像素，其 (y−f) 期望 = (p−f)，与歧义质量 m_δ 相乘构成下界主项。
- 这步把「sharpening 量 s」翻译成「calibration error 下界」。技术核心 = 选对 g*（沿 sign(p−f) 方向的 Lipschitz 函数）。

### Theorem —— Trade-off 形式化下界（主定理）
> 组合 Lemma 1（量化版）+ Lemma 2：
> **Dice(f) ≥ α  ⟹  smCE(f) ≥ Φ(α; m_δ, δ)**，
> 其中 Φ 关于 α 单调增、关于歧义质量 m_δ 单调增，且 **m_δ→0 时 Φ→0**（无歧义数据无 trade-off，符合直觉与 KiTS 上 DSC 几乎不降的实证）。

- 形态目标（待精化）：Φ ~ c · m_δ · s(α)，s(α) 为达到 Dice α 所需的最小 sharpening。
- 退化检查：α 低 → s(α)=0 → Φ=0（不逼高 Dice 就无需 sharpen，可校准）；m_δ=0 → 数据确定 → 无 trade-off。两个 sanity check 都过 ✓。

## 3.5 P0 玩具命题首推（主线亲算，验证 g* 构造）

> 目标：在最简情形把「sharpening ⟹ smCE 下界」严格走通，验证 Lemma 2 的 g* 构造法可行。结论：**走通了**（在 smCE 定义成立的前提下，无需任何待核常数）。

**P0 设定**。二分类逐像素，像素分两类：
- **确定区**（质量 1−m）：p(x)∈{0,1}，真值无歧义。
- **歧义区 𝓐**（质量 m）：p(x)=1/2（最大 aleatoric 歧义，模拟边界）。
预测器在 𝓐 上输出 f(x)=1/2+s（s∈[0,1/2] 为 **sharpening 量**，s=0 完全校准、s=1/2 完全 sharp）；在确定区输出 f=p（那里不影响，取最优）。

**Claim P0**：smCE(f) ≥ m·s。

**证**。由定义 smCE(f) = sup_{g:1-Lipschitz,→[−1,1]} E[g(f(x))·(y−f(x))]。
取测试函数 g*，令其在 f 在 𝓐 上的取值 v₀=1/2+s 处等于 −1，在确定区 f 取值处等于 0，中间按 1-Lipschitz 线性过渡（只要 |v₀ − {0,1}| ≥ s 即可容纳过渡，s≤1/2 时成立）。则：
- 确定区贡献：g*=0 → 贡献 0。
- 歧义区贡献：在 𝓐 上 E[y|x]=p=1/2，f=1/2+s，故 E[y−f | x∈𝓐] = 1/2 − (1/2+s) = −s；g*(v₀)=−1。
  贡献 = g*(v₀)·E[y−f|𝓐]·m = (−1)·(−s)·m = **m·s**。
合计 E[g*(f)(y−f)] = m·s，而 smCE 是 sup ≥ 此特定 g* 的值，故 **smCE(f) ≥ m·s** ∎。

**退化检查**：s=0（完全校准）→ 下界 0 ✓；m=0（无歧义）→ 下界 0 ✓。两个 sanity check 过。

**与 Dice 的方向性连接（P0 定性版）**。在 𝓐 上 p=1/2，真值 y 各半。期望硬 Dice 由「决策果断度」驱动：
- s→1/2（f→1，阈值后 ŷ=1）：抓住 𝓐 中全部真前景（TP），但也吃 FP；相对 soft 居中预测，overlap 型 Dice 期望更高（RankSEG：Dice-Bayes 是硬阈值）。
- s=0（f=1/2 居中）：阈值 τ=0.5 处摇摆，期望 Dice 不优。
→ **追高 Dice 逼迫 s↑ → 由 Claim P0 直接 smCE↑**。trade-off 在 P0 闭环成立（定性）。

**P0 留口（=P2 的硬骨头）**：把「Dice≥α」精确翻译成「所需最小 s(α)」的**定量映射**仍缺（D1 的 U-statistic 分母耦合 + D2 的 approximate 版 Lemma 1）。P0 只证了「s>0 ⟹ smCE>0 且随 s 单增」与两端退化，未给闭式 Φ(α)。**这是 P0→P2 的核心增量，也是论文主定理的技术核。**

**升级（主源已核 → P0 下界传到真·校准距离）**。[2211.16886] §7 定义 smCE_𝒟(f) := sup_{w∈L} E[w(v)(y−v)]，L=1-Lipschitz [0,1]→[−1,1]，**与上证逐字一致**（g* 即取 w∈L，无常数歧义）。其一致性定理 **½·d̲CE ≤ smCE ≤ 2·d̲CE**（d̲CE = lower distance to calibration，Def 5.1，真·ℓ1 距校准距离）给出：
> **d̲CE(f) ≥ ½·smCE(f) ≥ ½·m·s**。
即 trade-off 下界不止对 smCE，**对信息论 ground-truth 的校准距离同样成立**——P0 不再依赖度量选择，结论变强。`【已核主源】`

## 3.6 P1 —— 校准预测器的 soft-Dice 上确界 = 数据 aleatoric 决定（已证）

> 这步把 P0 的玩具升级为**任意 p 分布**的干净定理：在「输出真实概率」的校准约束下，soft-Dice 能到的最高值由数据歧义封死。

**建模假设 (A1，large-image 极限)**。单张大图、像素近 i.i.d. 抽样，由 LLN，soft-Dice 的分子分母各自集中，soft-Dice → **population soft-Dice**
> 𝔇(f) := 2·E[f(x)y(x)] / (E[f(x)] + E[y(x)]) = 2·E[f·p] / (E[f] + E[p])，
用到 E[y|x]=p(x)。记 B=E[f]，c=E[p]>0（有前景），R=E[fp]，Q=E[f²]。

**校准定义**（canonical）：f calibrated ⟺ E[p(x) | f(x)=v] = v。

**引理 1.1**：f calibrated ⟹ (i) E[f]=E[p]=c；(ii) E[fp]=E[f²]。
*证*：(i) E[f]=E[E[p|f]]=E[p]。(ii) E[fp]=E[f·E[p|f]]=E[f·f]=E[f²]。∎

**引理 1.2（calibrated 的 soft-Dice 闭式与上确界）**：f calibrated ⟹ 𝔇(f)=E[f²]/E[p]，且
> **𝔇(f) ≤ E[p²]/E[p]，max 唯一达到于 f=p。**
*证*（逻辑链已按 reviewer 意见修正——钉死极大点靠 calibration，不是 C-S）：由引理 1.1，分母=2c，𝔇=E[f²]/c。
- 上界：E[f²]=E[fp]（引理1.1(ii)）≤ √(E[f²])·√(E[p²])（C-S）⟹ E[f²] ≤ E[p²]，故 𝔇 ≤ E[p²]/E[p]。
- 极大点唯一性：C-S 取等 ⟺ f=λp（a.e.）；代入 calibration 约束 E[f]=E[p]（引理1.1(i)）得 λ·E[p]=E[p]，因 c=E[p]>0 ⟹ **λ=1 ⟹ f=p**。即「C-S 给上界，calibration 把比例常数钉到 1」。
- f=p 确在 calibrated 集合内（E[p|p]=p trivially）。∎
（σ-代数说明：f 为 x 的确定函数，σ(f)⊆σ(x)，由塔性质 E[p|f]=E[E[y|x]|f]=E[y|f]，故 calibration 写成 E[p|f]=f 与 E[y|f]=f 等价。）

**定理 P1（校准最优 = aleatoric 封顶）**：
> 在所有校准预测器中，soft-Dice 上确界 = **𝔇_cal\* = E[p²]/E[p]**，由 honest f=p 达到。
> 其与绝对上限 1 的**头空间（headroom）** = 1 − E[p²]/E[p] = **E[p(1−p)] / E[p]** = (标签方差均值)/(前景均值) = **归一化 aleatoric 不确定性**。

**含义**：超过 𝔇_cal\* 的那部分 Dice，**只能靠牺牲校准买**；且这块「可买头空间」精确等于数据的 aleatoric 不确定性 E[p(1−p)]。数据确定（p∈{0,1}）⟹ E[p(1−p)]=0 ⟹ 头空间=0 ⟹ **无 trade-off** ✓（对上 KiTS 实证几乎不损 Dice）。歧义越大头空间越大 ✓（对上 BraTS 损最多）。

## 3.7 P2 —— Calibration–Dice trade-off 的定量下界（主定理，**已修致命洞**）

> P1 给「超过 𝔇_cal\* 必失校准」（定性）。P2 给**定量**：超出多少 Dice，至少付多少校准误差。
> **修订记录**：reviewer 指出原引理 2.2（用 Błasiok d̲CE coupling 找附近校准 g）致命——coupling 的 g 活在扩展空间、无原始 p(x)，E[g·p] 无定义。**已改为原空间条件均值重标定，绕开 coupling，且结论直接落到实用的 ECE（顺带修 reviewer 的度量 overclaim）。**

**校准误差度量（用实用的 ℓ1-ECE，population/binless 版）**：
> ECE₁(f) := E| f(x) − E[p | f(x)] |。  （= 预测值与「同 f 值处真实频率」之差的平均，practically reported 的 ℓ1 校准误差）

**引理 2.1（soft-Dice 对预测的 Lipschitz 性，常数修正为 3）**：对任意 f, g（E|f−g|≤Δ），c=E[p]>0、分母≥c：
> **|𝔇(f)−𝔇(g)| ≤ 3Δ/c。**
*证*（写全差分，按 reviewer 意见）：记 R=E[·p], B=E[·]。
|𝔇(f)−𝔇(g)| = |2R_f/(B_f+c) − 2R_g/(B_g+c)|
≤ 2|R_f−R_g|/(B_f+c) + 2R_g|B_f−B_g|/((B_f+c)(B_g+c))。
- 第一项：|R_f−R_g|≤E[|f−g|·p]≤Δ（p≤1）⟹ ≤ 2Δ/c。
- 第二项：2R_g/(B_g+c)=𝔇(g)≤1（Dice≤1），余 |B_f−B_g|/(B_f+c) ≤ Δ/c ⟹ ≤ Δ/c。
合 ≤ 3Δ/c。∎

**引理 2.2′（原空间条件均值重标定 —— 替换致命的旧 2.2）**：令 **g(x) := E[p | f(x)]**。则
- (i) g 在原空间、为 f 的确定函数，**天然校准**：σ(g)⊆σ(f) ⟹ E[p|g]=E[E[p|f]|g]=g；且 E[g]=E[p]=c。
- (ii) **E|f−g| = ECE₁(f)**（按定义）。
- (iii) g 满足 P1 前提，且 𝔇(g)=E[g²]/c ≤ E[p²]/c（条件 Jensen：E[g²]=E[E[p|f]²]≤E[E[p²|f]]=E[p²]）。
*证*：(i)(ii) 见定义与塔性质；(iii) 条件 Jensen。∎ **全程原空间，无 coupling，洞已补。**

**定理 P2（trade-off 定量下界，主定理）**：取 g=E[p|f]，Δ=E|f−g|=ECE₁(f)，由引理 2.1 + 2.2′ + 定理 P1：
> 𝔇(f) ≤ 𝔇(g) + 3·ECE₁(f)/c ≤ E[p²]/E[p] + 3·ECE₁(f)/E[p]。
> 移项 ⟹ **ECE₁(f) ≥ (E[p]/3)·( 𝔇(f) − E[p²]/E[p] )**。
> 故 **𝔇(f) ≥ α ⟹ ECE₁(f) ≥ Φ(α) := (E[p]/3)·( α − E[p²]/E[p] ) = (1/3)( α·E[p] − E[p²] )。**

**性质核对**：
- Φ(α)>0 ⟺ α > E[p²]/E[p] = 𝔇_cal\*（只在「想超过校准最优」时才有代价）✓
- α≤𝔇_cal\* ⟹ Φ=0（不贪 Dice 可完美校准）✓
- 数据确定 E[p(1−p)]=0：E[p²]=E[p]，α≤1=𝔇_cal\* 恒成立 ⟹ Φ=0，无 trade-off ✓
- 度量诚实：本界对 **ECE₁（实用度量）直接成立**；smCE 版由 smCE≤ECE₁ 得（同向，更弱），不再 overclaim「对所有度量干净成立」。

## 9. E1 执行 —— Minimax 信息论下界（**🔴 两轮构造均被否决，现判 OPEN；正确方向=ECE-约束 minimax，见下**）

> 目标：把「对给定 f 的存在性下界」升级为「**任何学习器**在 worst-case 下都逃不掉的下界」。这是脱离 refinement 分解、上 COLT 档的关键——minimax 下界**不是**任何分解的推论。
> 核心张力：要超过校准最优 𝔇_cal\* 必须**硬承诺**（f→0/1）；但在最歧义处（p≈1/2）「往哪边承诺」正是有限样本最难估的；承诺错 = Dice 不升反降 + ECE 爆。

**两点构造**。特征 x~Unif[0,1]，固定符号模式 g:[0,1]→{±1}。两个分布：
> P₊：p(x)=½+ε·g(x)；  P₋：p(x)=½−ε·g(x)。  标签 y|x ~ Ber(p(x))。
学习器见 n 个 i.i.d. 标注像素，输出 f̂。两分布的 aleatoric 几乎相同（都 ≈½），仅「前景往哪偏」相反。

**引理 9.1（不可区分性，Le Cam）**：单样本 KL(Ber(½+ε)‖Ber(½−ε)) = 2ε·log((½+ε)/(½−ε)) ≤ 8ε²（小 ε）。故 n 样本 KL(P₊ⁿ‖P₋ⁿ) ≤ 8nε²，Pinsker ⟹ **TV(P₊ⁿ,P₋ⁿ) ≤ 2ε√n**。取 **ε = 1/(4√n)** ⟹ TV ≤ ½ ⟹ 任何「判 P₊还是P₋」的测试错误率 ≥ (1−TV)/2 ≥ ¼。

**⚠️ 自检发现 + 修正（prevalence=½ 退化）**：若构造里前景占比 c=E[p]=½（如上节朴素两点），则「全预测前景」f≡1 不用任何样本即得 𝔇=2c/(1+c)=2/3 > 𝔇_cal\*≈½——它靠**前景普遍性**而非 resolve 歧义白拿 Dice-excess（代价是巨大 ECE，被 P2 覆盖，但**击穿了「无样本无法获超额」的 minimax 叙事**）。reviewer 的 general-B 警告命中此处。**修法：必须用低 prevalence（小病灶）构造**，使 f≡1 因海量 FP 而废，获 Dice 才真需定位+样本。**此修正把 E1 与 E2 统一**（都由「落在候选区的有信息样本数」控制）。

**🔴 reviewer-2 二次否决：低 prevalence「修正」没堵住洞，E1 现判 OPEN（诚实降级）**。
失败构造（曾以为修好）：背景 p=0 + 候选区 2β、p=½+εgσ。**退化预测器**：「候选区全铺前景」f=1@候选/0@背景 → Dice=2β/(3β)=**2/3 > 𝔇_cal\*≈0.505**，excess≈1/6 = **O(1) 且 σ-无关、零样本**（候选区位置是公开信息，隐藏的 σ 只值 O(ε)）。第一轮 f≡1 漏洞只是搬了家。
> **根因（reviewer 点透）**：Dice-excess 主项来自「在哪铺前景」（位置/普遍性，公开），不来自 resolve 隐藏 σ。把 σ 放在「候选区内偏向」这个 O(ε) 维度是错的。

> **撤回过度声称**（违红线「不把弱结果当强卖」）：原「定理 E1 严格 / COLT 级 / minimax 已建立」**全部撤回**，E1 = **OPEN**。

**正确重做方向（reviewer-C，待证，不得当已成）**：
1. **σ 控位置，不控偏向**：让隐藏态决定「前景所在的离散假设」，使任何不知 σ 的预测器（含「铺满可疑区」）**都无法超过 𝔇_cal\***——excess 的唯一来源是猜中 σ。
2. **关键洞察**：纯「能否获 excess」minimax 必失败，因 P2 已说**牺牲校准总能换 excess**（flooding 即 ECE≈½ 换 Dice）。故正确 minimax 必须**约束校准**：

## 10. E2 执行 —— 去 A1：有限-N 真 Dice（U-statistic concentration，已推骨架）

> 去掉 A1（large-image 极限）。真 Dice = 2·Σf_iy_i /(Σf_i+Σy_i) 是 ratio-of-sums（U-statistic），分子分母同期望耦合。

**引理 10.1（ratio 集中）**：N 像素 i.i.d.，记 U=⅟N Σf_iy_i、V=⅟N Σ(f_i+y_i)，则 E[U]=E[fp]、E[V]=E[f]+E[p]=:S。Hoeffding：w.p. 1−δ，|U−E[U]|, |V−E[V]| ≤ √(log(4/δ)/(2N))=:η_N。当 **S ≥ 2η_N**（即分母不退化），D=2U/V 在 𝔇=2E[U]/E[V] 附近 Lipschitz：
> **|D − 𝔇(f)| ≤ (4/S)·η_N = O( 1/(S√N) ) w.p. 1−δ。**

**定理 E2（有限-N trade-off）**：当**期望前景规模够大** N·S ≳ log(1/δ)/η² 时，w.p. 1−δ 真 Dice 满足 |D−𝔇(f)|≤η，代入定理 P2：
> **ECE₁(f) ≥ (E[p]/3)·( D − E[p²]/E[p] ) − O(η)。**
即 A1 在「期望前景像素数 N·E[p] 足够大」时由集中**自动成立**，主定理对真 Dice 仅差 O(1/(S√N)) 修正。

**引理 10.2（控制参数 = 期望前景像素数 Nc，用方差版集中锐化）**：小病灶时 E[f]≈c（校准预测器 f≈p），S≈2c；分子分母方差由前景主导，Var(Σy_i)=N·E[p(1−p)] ≤ Nc，Var(V/N)=O(c/N)。故 V/N 的**相对波动** = √(Var)/均值 ~ √(c/N)/(2c) = **1/(2√(Nc))**。
> ⟹ ratio 的相对误差与 Jensen 偏差均为 **Θ(1/√(Nc))**（偏差 O(1/(Nc))）。**控制一切的是 Nc = 期望前景像素数，不是 N。**

**诚实定位（reviewer 的真难点，已锐化阈值）**：
- ✅ **Nc ≫ 1**（前景像素够多）：引理 10.2 ⟹ 相对误差 O(1/√(Nc))→0，E2 搞定，A1 可去，定理 E2 成立。
- ⚠️ **Nc = O(1)**（小病灶，前景仅数像素，临床 3mm 结节典型）：(i) Jensen 偏差 E[D]−𝔇 ~ Θ(1/(Nc)) 不可忽略且**系统性向下**（ratio 凸性，伤小病灶 Dice）；(ii) 集中失效，D 本身重尾。population 中介崩，须**直接对随机变量 D 证 high-prob 下界**。
  - **攻法（论文主攻，OPEN）**：把前景像素数 K~Binomial(N,c)≈Poisson(Nc) 显式条件化；给定 K，D 的分布可写成超几何/Poisson-binomial 泛函；目标证「E[D | 小 K] 要高 ⟹ 在 K 个前景上硬承诺 ⟹ 这些像素 ECE 大」的有限组合下界。工具候选：self-normalized 集中（de la Peña）、Efron–Stein 方差界、或直接 Poisson 化后逐 K 求和。
  - 这是 trade-off **最强、最值钱**处（小病灶正是临床最在意、aleatoric 边界占比最高）。与 E1 的 1/√n minimax floor 同源：**样本/前景稀缺 ⟹ 信息论不可避免代价**。可把 E1 的两点构造移植到「前景像素数」维度，得统一的「稀缺 ⟹ floor」叙事。

### 10.1 小病灶问题的精确归约（well-posed OPEN，主线已做）

> 不硬造定理；把 OPEN 收成一个可直接攻的条件化问题，供后续/合作者起手。

**条件化于前景数 K**。前景集 S，|S|=K~Binomial(N,c)≈Poisson(λ)，λ=Nc。预测正类集 Ŝ，|Ŝ|=m。硬 Dice：
> **D = 2|Ŝ∩S| / (m+K)**，K=0 时约定 D=1[m=0]（空真值边界，须单独处理——临床「无病灶」图）。
给定 K，|Ŝ∩S| 服从超几何（随机 Ŝ）或由 p 决定的 Poisson-binomial（最优 Ŝ）。

**精确目标命题（OPEN）**：存在常数 κ(λ)>0 使任何预测器满足

> reviewer 最致命的非技术批评：P1 的 headroom=E[p(1−p)]/E[p] 就是 Bröcker/DeGroot-Fienberg refinement 分解的 resolution/uncertainty 项，1980s 已知。**不能装看不见。** 这里把「哪些是换皮、哪些是真贡献」划清。

**诚实让出（但比 reviewer 担心的轻，researcher 三方核 Bröcker 后精化）**：在 A1（population）下，1−𝔇_cal\*=E[p(1−p)]/E[p]，其中 **E[p(1−p)] = irreducible aleatoric Brier**（=完美预测器 f=p 的 Brier = UNC−最大 RES；q̄(1−q̄)−Var[p]=E[p]−E[p²]=E[p(1−p)]）。
> ⚠️ 注意：E[p(1−p)] **不是** Bröcker 的 resolution 项（RES=Var[E[Y|p]]），而是**不可约 aleatoric 残差**。故 reviewer 说的「P1=resolution 换皮」**措辞不准**——P1 确与已知 aleatoric 量相关，但 𝔇 的 **ratio 归一化 E[p(1−p)]/E[p]**（除以 prevalence）不是任何标准分解项，是 Dice 特有。**保守表述**：P1 与已知 aleatoric Brier 同源，**不单独当主贡献**，但也非简单换皮。正文把 P1 作 setup。

**真贡献（分解给不出的三件）**：
1. **跨「非 proper 度量」的桥**：Bröcker 分解 E[S]=REL−RES+UNC **只对 proper score S 成立**。而 Dice **非 proper**（RankSEG Lemma 9：连 CE/focal 都非 Dice-calibrated）。「一个**非 proper** overlap 度量的校准最优天花板竟仍等于 proper-score 量 E[p²]/E[p]」——这道桥不是分解的推论，是 Dice 的 ratio 结构 + E[fp]=E[f²]（仅在校准时）撑出来的。分解理论对非 proper 度量沉默。
2. **P2 是不等式，不是恒等式**：refinement 给的是**等式分解**（把一个 proper score 拆成三项）。P2 给的是 **ECE₁(f) ≥ (E[p]/3)(𝔇(f)−𝔇_cal\*)** —— 用**非 proper 的 Dice 超额**下界**预测器的 ECE**。这是「度量 A 的值 ⟹ 度量 B 的下界」的跨度量不等式，且斜率 E[p]/3 来自 Dice 分母 1/(B+c) 的归一化结构，分解式里没有这东西。
3. **E1 minimax 完全在分解之外**：定理 E1 的 1/√n 是 **finite-sample、信息论、方法无关**下界。refinement 是 population 恒等式，**永远给不出任何 n-依赖**。这是 COLT 档贡献的核心，与分解零关系。

**一句话定性**：P1 作 setup（与已知 aleatoric Brier 同源，但 ratio 归一化非标准分解项），**主贡献 = P2 跨非-proper-度量 trade-off 不等式 + E1 的 1/√(nβ) minimax floor + E2 小病灶 OPEN**。正文 Intro 主动写「我方 ≠ refinement 分解」（§11 三条），把 P1 降 setup，主定理挂 P2+E1。审稿人挥不出「trivial」这一刀。
