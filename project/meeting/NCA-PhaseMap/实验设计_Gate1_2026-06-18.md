## 🔴-6 全局修正：CLIP_NORM 非官方自加，no-clip 改主条件

**核实**：官方 `M3D-NCA-official/src/agents/Agent.py` L102-103 = `loss.backward(); optimizer.step()`，全 `src/` 零 `clip_grad_norm_`。distill 2020 用 per-variable L2 norm（另一机制）。G5 三重实证全带 `CLIP_NORM=1.0` = 违复现零偏离红线。

**致命点（动摇支柱2）**：clip=1.0 下优化器用裁到 norm≤1.0 的梯度，名义 `max_grad_norm` 记 396 但真实步长被夹平 → 「名义 max_grad 与塌缩无关 r=0.238」是 clip artifact 必然，非「梯度真无关」。A3 必在 no-clip 下重测。

**处置**：主条件=官方 no-clip（主结果以此为准）；clip=1.0=对照（解释 G5）；腿①B1/B2 + 腿②G* 加 clip 维度；腿①-b/腿③ 用 no-clip。
**no-clip 归因预案**：①no-clip ur* 与 G5 一致→相变稳健于 clip，G5 可沿用 ②漂移/消失→G5 临界部分由 clip 诱导,主结论改 no-clip 实测,G5 标注修正 ③出真发散(loss→NaN,diverged=1)→严区分发散≠塌缩,diverged 单列不计 collapse,A3 只在「收敛但塌缩」run 做。

**collapse 判据（冻结）**：
```
collapse(run) := (not diverged) and final_dice < max(0.01, dice_bg + 3·σ_bg)
  diverged(loss→NaN) 单列不计 collapse(塌缩≠发散)。
  Hippo: dice_bg≈0→阈≈0.01(不破复现)。BraTS: dice_bg+3σ 自适应抬阈防低前景假 collapse。
```
**前景控制**：BraTS 切片优先保留前景占比与 Hippo 可比(排除<2% 极低前景约12%)。
**真/假 KILL**：①高 ur 仅≈dice_bg→假 KILL 回查数据 ②高 ur 显著>dice_bg、越临界跌回→真相变按 K1 判 ③全程>baseline 无断崖→真 KILL 临界消失。
**诚实回退**：B0 标定后仍无法分离 collapse/低分→A4 降「第二脑 MRI 复现」,headline 不写「普适」。

## 腿③ — R1 机制探索（修 🟡-3+🟡-7,预期大概率 K2）
