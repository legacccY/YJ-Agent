## 一、世界模型是什么

**定义**：一个内部模型，学环境的**压缩潜在状态 + 动力学**，能在「给定当前状态 + 动作」时预测下一时刻状态（及奖励/终止），并能向前连续展开（rollout）。三个结构性特征缺一不可：**状态演化（dynamics）、动作条件（action-conditioned）、可 rollout**。

**与普通生成模型的本质区别**：

| | 生成模型（GPT/Diffusion） | 世界模型 |
|---|---|---|
| 预测目标 | 原始像素/token | 潜在**状态**的演化 |
| 动作 | 无（或 prompt 当动作） | 显式输入，驱动状态转移 |
| rollout | 非设计目标 | 核心功能，可想象 N 步 |
| 目的 | 拟合数据分布 | 为规划/决策服务 |

**白话**：脑子里的沙盘。不用真走出去试每一步，在脑中「想象」——「左转三步后到哪」——沙盘按运动规律算给你。它不记每粒沙（不还原所有细节），只抓规律。

**谱系根**：
- Ha & Schmidhuber 2018《World Models》（VAE+MDN-RNN+Controller 三件套 + 在「梦境」中训练策略）= 现代旗舰原型。[arXiv:1803.10122](https://arxiv.org/abs/1803.10122)
- 更早根：Sutton 的 Dyna（1991，「规划就是在脑中试做」）、Kalman 滤波（1960，线性状态空间）、Friston 自由能原理（大脑即预测机，[PMC7468257](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC7468257/)）、Jay Wright Forrester 的 mental model（1971）。
- LeCun 2022《A Path Towards Autonomous Machine Intelligence》把世界模型设为通往 AGI/具身智能的关键——具身体必须「在脑中试错」而非每次真交互。[OpenReview BZ5a1r-kVsf](https://openreview.net/pdf?id=BZ5a1r-kVsf)

**关键概念词典**：
| 术语 | 一句话定义 |
|---|---|
| Latent dynamics | 压缩潜空间中描述状态如何随时间和动作演化的规律，非像素/token 层面 |
| Rollout | 给定初态+动作序列，在 WM 内部连续预测未来，无需真实交互 |
| Imagination/Dreaming | 在内部模型生成的虚拟轨迹中推演，用于策略训练/规划 |
| Action-conditioned prediction | 下一状态显式依赖动作 a，即 P(s_{t+1}\|s_t,a_t)，区别纯预测模型的核心 |
| Counterfactual / do-演算 | 模拟「若当时做 X 而非 Y，结果如何」，不需真实发生 |
| Planning in latent space | 在低维潜表示做多步规划/搜索，而非原始观测空间 |

## 二、架构家族（四大流派）

### 1. RSSM 系（PlaNet / Dreamer，Hafner）
状态 = 确定性 h_t（GRU 记忆）+ 随机 z_t，**在潜空间想象 + actor-critic**。
- PlaNet（ICLR 2020）：首个纯潜空间规划，CEM 规划器。[arXiv:1811.04551](https://arxiv.org/abs/1811.04551)
- Dreamer V1（连续 z）→ V2（离散 categorical z，超 Rainbow）→ V3（Symlog + KL 平衡 + free bits，单套超参跨 150+ 任务，首个从零挖 Minecraft 钻石，2025 上 Nature）。[arXiv:2301.04104](https://arxiv.org/abs/2301.04104) / [Nature](https://www.nature.com/articles/s41586-025-08744-2)
- Dreamer 4（2025）：纯离线视频-动作数据完成 Minecraft 钻石。
- 优点：样本效率最高；缺点：rollout 误差累积。

### 2. JEPA 系（I-/V-JEPA，LeCun）
**不重建像素，在表示空间预测**。主动丢弃不可预测噪声。
- I-JEPA（图像，[arXiv:2301.08243](https://arxiv.org/abs/2301.08243)）、V-JEPA（视频，效率提升 1.5-6×）。
- V-JEPA 2（2025，[arXiv:2506.09985](https://arxiv.org/abs/2506.09985)）：300M Transformer，100 万小时视频 + 62h 机器人数据 → zero-shot 机械臂规划，比 Cosmos 快 30×。

### 3. 生成式 / Diffusion 系
世界模型 = 高保真视频生成器，动作当条件。
- GameNGen（神经 DOOM 引擎，20FPS，[arXiv:2408.14837](https://arxiv.org/pdf/2408.14837)）
- DIAMOND（NeurIPS24 Spotlight，score-matching 扩散，Atari 视觉保真最高，HNS=1.46，[arXiv:2405.12399](https://arxiv.org/abs/2405.12399)）
- Genie 1（11B，无监督 LAM 学潜在动作，[arXiv:2402.15391](https://arxiv.org/abs/2402.15391)）/ Genie 2（自回归潜扩散，最长 1 分钟 3D）/ Genie 3（720p/24fps/数分钟一致性 + 对象永久性，2025-08）
- Oasis（ViT 自编码器 + DiT，20FPS 实时 Minecraft）、NVIDIA Cosmos（物理 AI 世界基础模型，MoT 架构）

### 4. Token / 自回归系（IRIS）
VQ 把帧切 token，Transformer 自回归预测，「图像当语言」。Atari 100k SOTA。[arXiv:2209.00588](https://arxiv.org/abs/2209.00588)。改进版 Δ-IRIS 编码帧间增量。

### 横向特性对比
| 流派 | 预测目标 | 潜变量 | 动作条件 | 长程一致性 | 算力 | 可规划性 |
|---|---|---|---|---|---|---|
| RSSM/DreamerV3 | 潜空间 | 确定 h+随机 z（离散） | 是 | 中 | 单卡可训 | 高（latent imagination） |
| V-JEPA 2 | 表示空间 | predictor latent | V-JEPA2 加入 | 高 | 大规模预训练 | 中 |
| GameNGen/DIAMOND | 像素（扩散） | 无 | 是 | 中 | 中-大 | 低 |
| IRIS | 离散 token | 离散 token | 是 | 中 | 中 | 中 |
| Genie 2/3 | 像素（AR 潜扩散） | 无 | 是 | 中高（离屏记忆） | 大 | 低 |

### 核心技术挑战
① compounding error（误差级联，见 §七.8）② 长程记忆/一致性（实测 9 步后 90%→30%）③ 3D/物理一致 ④ 评估难（FVD/FID 与下游性能弱相关）。

## 三、有什么用（应用领域）

- **自动驾驶**（落地最快、资本最密）：Wayve GAIA-1/2/3（[arXiv:2503.20523](https://arxiv.org/html/2503.20523v1)）、DriveDreamer（ECCV24）、Vista（NeurIPS24）、占用世界模型——**生成稀有危险场景做安全验证**。
- **游戏/交互世界生成**：Genie 3、GameNGen、Oasis——「神经网络替代游戏引擎」。

## 七、★ 数学理论完整推导

从第一性原理推一遍：世界模型在数学上是什么、为什么这么训、为什么会崩。

### 7.1 设定：POMDP / 状态空间

环境 = 部分可观马尔可夫决策过程。隐状态 s_t、观测 o_t、动作 a_t、奖励 r_t：

```
转移： s_t ~ p(s_t | s_{t-1}, a_{t-1})
观测： o_t ~ p(o_t | s_t)
奖励： r_t ~ p(r_t | s_t)
```

智能体看不到 s_t，只看 o_t。一条轨迹的联合生成模型：

```
p(o_{1:T}, r_{1:T}, s_{1:T} | a_{1:T}) = ∏_t p(s_t|s_{t-1},a_{t-1}) · p(o_t|s_t) · p(r_t|s_t)
```

**世界模型 = 学这个因式分解里的转移核 p(s_t|s_{t-1},a_{t-1}) + 观测/奖励头。** 其余所有架构差异，都是「怎么参数化这个转移核 + 在什么空间预测」的不同选择。

### 7.2 为什么要潜变量 + 变分推断

真后验 p(s_{1:T}|o_{1:T},a) 不可解（高维积分）。引入变分后验（编码器）：

```
q(s_{1:T}|o_{1:T},a_{1:T}) = ∏_t q(s_t | s_{t-1}, a_{t-1}, o_t)
```

推 ELBO，用 Jensen。注意 Dreamer 最大化的是**观测 + 奖励的联合似然**（奖励 r 当作一路被建模的「观测」），故左边是 p(o,r|a) 而非只 p(o|a)：

```
log p(o_{1:T}, r_{1:T} | a) = log ∫ p(o,r,s) ds
                            = log E_q[ p(o,r,s) / q(s) ]
                            ≥ E_q[ log p(o,r,s) − log q(s) ]          (Jensen)
```

代入因式分解，逐项телескоп展开：

```
log p(o,r,s) − log q(s)
 = Σ_t [ log p(o_t|s_t) + log p(r_t|s_t) + log p(s_t|s_{t-1},a_{t-1}) − log q(s_t|·) ]
```

对 s_t 取 E_q，后两项合成 KL：

```
ELBO = E_q Σ_t [ log p(o_t|s_t)      ← 重建（观测解码）
                + log p(r_t|s_t)      ← 奖励预测
                − KL( q(s_t|s_{t-1},a_{t-1},o_t) ‖ p(s_t|s_{t-1},a_{t-1}) ) ]  ← 动力学先验
```

**这就是 PlaNet/Dreamer 的世界模型训练目标。** 关键在 **KL 项**：它逼「只靠历史+动作（没看 o_t）的先验」去逼近「看了 o_t 的后验」。先验学好了，就能在**没有真实观测**时凭历史向前编故事——**这正是 imagination/rollout 能成立的数学根**。

### 7.3 RSSM 参数化（为何拆 h + z）

把状态拆成确定性 + 随机两半：s_t = (h_t, z_t)。

```
h_t = f_φ(h_{t-1}, z_{t-1}, a_{t-1})        ← GRU，确定性记忆，可靠传递长程信息
先验 prior：    p_φ(z_t | h_t)               ← 不看观测
后验 posterior：q_φ(z_t | h_t, o_t)          ← 看观测
```

KL 只作用在 z_t 上。h_t 确定性 → 长程记忆不被随机性冲掉；z_t 随机 → 吸收不可预测部分。

**KL balancing（Dreamer 关键 trick，防塌缩）**，用 stop-gradient 解耦两个方向：

```
KL = α·KL( sg(q) ‖ p ) + (1−α)·KL( q ‖ sg(p) )，   sg = stop-grad
```

前项只训先验追后验，后项只训后验别离先验太远，互不反向污染。再配 free-bits（KL 设地板）防后验塌缩。

### 7.4 想象 + actor-critic（Dreamer 怎么「在梦里学策略」）

世界模型训好后，策略训练**完全在潜空间、零真实交互**。纯滚先验生成想象轨迹：

```
ŝ_{τ+1} ~ p_φ(· | ŝ_τ, a_τ)，     a_τ ~ π_θ(· | ŝ_τ)
```

定义价值 V_ψ(s)，用 **λ-return**（平衡偏差/方差）做回归目标：

```
V^λ_t = r_t + γ[ (1−λ)·V_ψ(ŝ_{t+1}) + λ·V^λ_{t+1} ]        (递归)
```

- **Actor**：最大化 E[ Σ V^λ_t ]。潜空间可微 → 直接对动力学反向传播（连续动作）；离散动作用 REINFORCE。
- **Critic**：回归 V_ψ → V^λ（stop-grad 目标）。

「在梦里学」= 整个策略梯度只过想象轨迹，真实环境只用来训世界模型本身。

### 7.5 规划视角（PlaNet / latent MPC）

不学摊销策略，改在线规划。每步解：

```
a*_{t:t+H} = argmax_a  E[ Σ_{τ=t}^{t+H} γ^{τ−t} r(ŝ_τ) ]
```

在想象 rollout 上用 CEM（交叉熵法）优化动作序列。**潜空间让搜索变廉价。** 本质：世界模型 + 奖励模型 = 一个**学出来的 MDP**，规划 = 在潜 MDP 里做近似动态规划（逼近 Bellman 最优）。

### 7.6 JEPA / 非生成路线（LeCun 为何砍掉解码器）

**丢掉 p(o_t|s_t) 解码器**，改在表示空间预测：

```
s_x = E_θ(x)，   s_y = E_ξ(y)        (y = 未来/被遮挡目标)
预测： ŝ_y = P(s_x, 隐变量)
能量： E(x,y) = ‖ P(s_x) − s_y ‖²    (在表示空间，不是像素)
```

目标 = 真 (x,y) 对上最小化能量。**致命问题：塌缩**——编码器把一切映成常数，能量恒 0，平凡解。防塌缩两招：

1. **非对称 + EMA（BYOL/I-JEPA）**：目标编码器 = 在线编码器滑动平均 ξ ← m·ξ + (1−m)·θ，目标侧 stop-grad。打破对称 → 避开常数不动点。
2. **方差-协方差正则（VICReg）**：显式维持各维方差（防坍成点）+ 去相关（防维度冗余）。

**为何非生成**：像素空间充满不可预测细节（树叶抖动），强迫重建=浪费容量学噪声、引入错误归纳偏置。表示空间预测专注「语义上可预测的结构」。对应 Friston 自由能/predictive coding——最小化**潜空间**预测误差。

### 7.7 Diffusion 世界模型（score-based）

把转移核建成条件扩散 p(o_{t+1} | o_{≤t}, a_t)。前向加噪 x_k = √ᾱ_k·x_0 + √(1−ᾱ_k)·ε，训去噪器预测噪声：

```
L = E_{k,ε,x_0} ‖ ε − ε_ψ(x_k, k, [过去帧, 动作]) ‖²
```

反向 SDE/ODE 采样 → 下一帧（DIAMOND）。优点：建完整分布（多模态）、保像素细节。代价：每帧采样贵、无显式可规划 latent。

### 7.8 Compounding error（为什么 rollout 越滚越烂——理论根）

世界模型最硬的理论极限。设单步模型误差 ε（TV 距离）。**телескоп引理**：对马尔可夫核 P̂（模型）vs P（真），h 步分布漂移满足

```
‖ P̂^h μ − P^h μ ‖_TV  ≤  Σ_{k=0}^{h-1} sup_s ‖ P̂(·|s) − P(·|s) ‖_TV  ≤  h·ε_max
```

误差**线性累积于 horizon**（每步预测被当下一步输入，误差反馈）。进一步，模型内策略的价值估计偏差（Janner MBPO / Ross-Bagnell 型界）约

```
|V_model − V_true|  ≲  ε · γ / (1−γ)²
```

返回-差随 horizon 近似**二次**放大。这就是为什么：imagination horizon 要短、用 λ-return 折中、分支 rollout、JEPA 在表示空间滚（误差量级更小）、Genie 2 加显式离屏记忆。

> 注：本台 C047 踩的 mode-collapse 是**另一层**失败模式（容量/稳定性），不是这个 compounding，但都属「rollout 不可信」家族。

### 7.9 统一视角（一句话收口）

> **世界模型 = 一个学出来的（潜）转移核 T_φ + 观测/奖励读出头。**
> - 训法二选一：**生成式**（最大似然 → ELBO，RSSM/IRIS/diffusion）或 **能量/对比式**（JEPA，表示空间预测 + 防塌缩）。
> - 用法 = 把 T_φ 沿 horizon **复合（rollout）** + 对预测奖励**优化动作/策略**（planning/RL）或直接**读表示**（下游）。
>
> RSSM、token、diffusion、JEPA 的全部差异，只是「T_φ 怎么参数化 + 在像素/token/潜变量/表示哪个空间预测」。
