## 2. 预登记判据（结果回来前锁定，跑后不许改）

### Pre-exp A（路线①，job 1451330 ncaj_probe_conv）
- ckpt：`logs/a2_scp_nca_vits_nih10k/jepa-ep50.pth.tar`（S16）｜数据：NIH held(`probe_test.txt`) vs VinDr 各 N=200｜**主臂锁 max_steps=16 不外推**（避训练分布外未定义动力学污染）
- 度量：每样本收敛步数 k* = earliest k where cos(pred_k,pred_{k-1})<0.01；两域 k* 直方图重叠（口径=Bhattacharyya 系数，跑前定死）
- NIH 用 held `probe_test.txt`（非 pretrain_10k）避「seen vs unseen」混淆

| 实测 | 结论 |
|---|---|
| 两域 k* 重叠 > 40% | 🔴 路线① **死**：收敛步数无域判别力 |
| 20–40% | 🟡 边际：需 ViT hidden state 对照才能判 |
| < 20% 且 VinDr k* 系统更大 | 🟢 路线① **活**：值得设计 ViT 信噪比对照 |
| 截断率（k*=16 占比）> 30% | ⚠️ 离散 k* 失效 → 改连续量 cos(pred_16,pred_15) 重判三档 |

### Pre-exp B（路线③，job 1451331 ncaj_probe_genesis）
- ckpt 同 A，S16，K=16 不外推｜NIH held 100 样本｜sim(k)=cos(pred_k,pred_16)｜按难度（coder 用 pred_K norm proxy，⚠️非真 loss，分组粗，解读注意）分易/中/难
- 形状平行口径：三组 sim(k) 各归一到 [sim(1),1] 后两两 RMSE<0.02 判平行（跑前定死）

| 实测 | 结论 |
|---|---|
| 三组形状平行（RMSE<0.02） | 🔴 路线③ **死**：轨迹=h16+插值噪声，无独特信息 |
| 困难组显著更慢/平台/振荡 | 🟢 路线③ **活**：轨迹含终点外结构 |
| 单调样本 >90% 且 osc≈0 | 单调吸引坐实（弱化③） |
| osc>0 样本 >30% | 推理轨迹振荡（支持 Kvalsund，标 latent≠像素需区隔，保一线） |

## 4. 结果落点

- **Pre-exp A（路线①）= 🔴 死**（analyst Bash/Grep 核 csv，2026-06-17）：conv_step 0/400 收敛（截断率 100% → 触发预登记替代量分支）。final_norm 两域 median NIH 102.72 / VinDr 102.27（差 0.44，Cohen's d=0.16 极小），**Bin overlap 73% / Bhattacharyya 0.901 / KS p=0.142 不显著** ≫ 40% 死亡阈。端点幅度两域完全混叠 → 收敛步数更不可能区分域。与 researcher 前提核查（scanner artifact baseline 天花板近 1.0）同向。替代量口径备注：非原定 cos(pred_16,pred_15)（脚本未存逐步 cos），但 final_norm 是更强负证据，**不需补跑**即可宣判。
- **Pre-exp B（路线③）= 🔴 死**（同上）：三组 sim(k) 归一后两两 RMSE = easy-med 0.0190 / easy-hard 0.0113 / med-hard 0.0089，**全 <0.02 平行阈** → 轨迹=h16+插值噪声，无独特信息。**100/100 单调、max_rebound 全精确 0.0、零振荡**。难度仅影响 k=1 初始投影（easy 0.74 vs hard 0.68），k≥4 后差值缩到 <0.005。**Kvalsund 裁决：latent NCA 无像素级振荡吸引子，单调直奔终态，Kvalsund(2604.10639) 不迁移**。
