SCP-JEPA：稳定细胞预测器驱动的可解释联合嵌入世界模型
从「NCA 替代 ViT」到「NCA 解锁 ViT 做不到的事」
配套：CheXWorld（CVPR 2024）+ MED-NCA（MedIA 2025）+ Med-NCA 复现报告
目标会议：CVPR / ICCV / NeurIPS
最后更新：2026-06-14
核心叙事（一句话版）
ViT predictor 给你一个黑盒预测。NCA predictor 给你预测 + 不确定性 + 可解释轨迹 + 随时推理。且在适当稳定化后，下游分数不输 ViT。
0.1 为什么不是原来的 NCA-JEPA 方案
原方案（NCA-JEPA pilot）验证的假设是「NCA predictor 比 ViT predictor 更好（省参 + 等变）」。经过批判性审视，这个假设存在三个结构性缺陷：
| 缺陷 | 详情 |
|---|---|
| 省参不是痛点 | JEPA 预训练成本大头在 ViT encoder（86M 参数，300 epoch，8 GPU），predictor 省 100× 参数对总成本影响≈0 |
| 等变优势前提存疑 | ViT + 充足旋转数据增强能学出近似等变性；CheXWorld 的 domain variation modeling 已经在做类似的事 |
| 稳定性是动力学本质问题 | 复现报告：0/11 前列腺跑满 epoch，同种子不同盆地。Spectral norm 降低放大因子但 128 步链乘法的指数敏感性仍在 |
结论：不要打「NCA 比 ViT 更好」这场可能打不赢的仗——打「NCA 能做到 ViT 做不到的事」这场仗。
0.2 NCA 不可替代的三张牌
这三张牌来自 MED-NCA 和复现报告的实证基础，不是凭空想象：
| 能力 | NCA predictor | ViT predictor | 证据来源 |
|---|---|---|---|
| 內建认知不确定性（Epistemic UQ） | 随机点火 + 10× 推理方差 → 天然质量信号 | ❌ 需额外贝叶斯头 | 复现报告 §5：NQM Spearman ρ=0.47，检出率 1.0 |
| 可解释推理轨迹（Interpretable rollout） | 每步通道状态可视化，展示预测如何"生长" | ❌ 注意力图粗粒度 | MED-NCA Fig.4：NCA-VIS 通道可视化 |
| 随时推理（Anytime prediction） | 迭代过程可提前终止，粗糙但快速 | ❌ 全或无 | NCA 的迭代本质（Growing NCA, Mordvintsev et al. 2020） |
这三张牌构成论文的 moat——即使下游分数略低于 ViT predictor（≤3 AUROC），审稿人也无法否认这些能力 ViT 没有。
本方案只回答三个问题（不多做）
Pillar 1 — 稳定性：经过系统稳定化（spectral norm + EMA anchoring + deterministic mode）的 NCA predictor，在 JEPA 框架下能否稳定训练、不塌缩、不出现双峰盆地？
Pillar 2 — 表示质量：稳定化后的 NCA predictor 在下游 linear probe 上是否不显著差于 ViT predictor（非劣效性检验，ΔAUROC > -3）？
Pillar 3 — 独有能力（Moat）：NCA predictor 的 (a) 方差是否编码有意义的预测不确定性？(b) 通道状态是否形成可解释的解剖结构？(c) 随时推理的步数-质量 trade-off 曲线是否平滑可用？
⚠️ 本方案不追 SOTA、不上 3D、不做多模态。只要信号 + 独有能力成立，不要终版数字。
⚠️ 复用 I-JEPA 官方代码库 + CheXWorld 预训练 pipeline，只换 predictor 模块。

方法框架：SCP-JEPA
3.1 整体架构
与 CheXWorld 共享 backbone（ViT-S, 预训练用 ViT-B），仅替换 predictor 模块：
Input → Context Encoder (ViT, EMA→Target Encoder) → NCA Predictor → Predicted Target Embedding
                                                              ↑
                                                       Latent z (mask/Δ/a)
NCA predictor 内部结构：
Input: context features h_x + mask tokens with PE
  ↓
For step = 1 to S:
  Perceive: Conv3×3(h) + identity → 局部邻域感知
  Update: fc₁(ReLU(fc₀(perceive))) → 残差增量
  Fire mask: Bernoulli(0.5) → 随机激活
  Residual: h ← h + fire_mask ⊙ update
  Re-inject: h[0] ← input_embedding (保持输入锚定)
Output: predicted target embeddings ĥ_y
3.2 稳定化三件套（理论依据）
问题本质：NCA predictor 是一个迭代动力系统 h{t+1} = ht + mt ⊙ fθ(h_t)。当 Jacobian 谱半径在某步超过 1，扰动指数放大。128 步（2 层 × 64 步）的链乘法使即使 ~10⁻⁷ 的 cuDNN 浮点扰动也能在终点造成 ~10² 的差异。
三件套：
| 组件 | 机制 | 理论依据 | 实现 |
|---|---|---|---|
| Spectral Normalization | 约束每步更新函数的 Lipschitz 常数 < 1 | 压缩映射原理：Banach 不动点定理保证迭代收敛 | torch.nn.utils.spectral_norm 包在 conv/fc 层 |
| EMA Target Anchoring | EMA target encoder 提供稳定预测目标，防止 predictor 追逐自身漂移 | 与 I-JEPA 原版一致，但在此处额外起到「稳定锚」作用——predictor 不直接拟合自身输出 | 复用 I-JEPA 的 EMA target，momentum 0.996→1.0 |
| Deterministic Mode | 训练时关掉 cuDNN benchmark、启 deterministic algorithms、固定 fire mask 的 torch.manualseed | 消除 cuDNN/atomic 不确定性，使种子级复现成为可能（复现报告 §4 的直接教训） | cudnn.deterministic=True, cudnn.benchmark=False, usedeterministic_algorithms(True) |
3.3 三个独有能力的形式化
能力 1：內建认知不确定性（Built-in Epistemic UQ）
给定输入 x，运行 predictor K 次（每次 fire mask 不同）
μy = (1/K) Σk gϕ(hx; z, m_k)
σy = std({gϕ(hx; z, mk)}_{k=1..K})
预测不确定性 = ||σ_y||₂  （per-sample predictor 输出方差）
关联验证：||σ_y||₂ vs 预测 L2 error 的 Spearman ρ。若 ρ ≥ 0.3 → 方差是有效的预测质量信号。
能力 2：可解释推理轨迹（Interpretable Rollout）
记录 predictor 内部通道状态 {h^(t)}_{t=1..S}（S 步迭代中每步的隐藏状态）
用 NCA-VIS 风格可视化每步的通道激活
目标：观察到从"模糊全局猜测" → "逐步细化局部解剖结构"的涌现过程
定性分析 + 可能的用户研究（2-3 位放射科医生评估轨迹是否具有解剖学意义）。
能力 3：随时推理（Anytime Prediction）
质量曲线：Q(k) = 在 k 步提前终止时的预测质量（vs 满步预测的 cosine similarity）
计算 Q(1), Q(2), Q(4), Q(8), Q(16), Q(32), Q(64)
目标：Q(k) 单调递增且前期快速收敛，展示"可牺牲精度换速度"的灵活性
对照臂设计（4 臂，控变量）
| 臂 | predictor | 步数 | 验什么 |
|---|---|---|---|
| A0 | ViT predictor（I-JEPA 原版） | 1（单次前向） | 基线：管线 sanity + 下游上限 |
| A1 | vanilla NCA predictor（无约束） | S=64 | 反例：NCA 直接搬入 JEPA 是否依然塌缩（预期：比分割任务稳但仍可能不稳） |
| A2 | 稳定 NCA predictor（三件套） | S=64 | Pillar 1+2 主臂：稳定化是否有效 + 下游是否不劣于 A0 |
| A3 | 稳定 NCA predictor（三件套，同 A2）| S=64 | Pillar 3 主臂：在 A2 的 checkpoint 上评估三能力（不确定性+可解释性+随时推理） |
A3 和 A2 是同一个模型，不做独立的训练臂——A2 训完后直接评估三能力。
所有 NCA 臂 predictor 参数预算 ≤ A0（ViT predictor 参数量）。
A1 可选（时间紧可砍），但保留能讲「问题→修复」的干净故事。

| 指标 | 定义 | 绿灯标准 |
|---|---|---|
| UQ 质量 | predictor 方差 vs 预测 L2 error 的 Spearman ρ | ρ ≥ 0.3, p < 0.05 |

Silent-bug 哨兵（强制，继承原 Pilot + 复现报告教训）
跑任何实验前，以下 8 条必须先过。任一红灯 = 管线 bug，先修再跑。
归一化 assert：log 喂入张量的 mean/std == 期望 NORM 统计；raw 值溜进来直接 raise
EMA sanity：target ≠ context（momentum 在动）；target encoder 不吃梯度
塌缩 canary：嵌入 std < 阈值 → flag（不是「loss 真好」）
overfit-one-batch：全跑前，模型须把单 batch loss 压到≈0；压不下 = 代码 bug，立即 abort
z-shuffle 对照：打乱 mask/条件 z → 下游应掉。不掉 = z 没接进去
Determinism 留痕：记录 cudnn.deterministic / benchmark / seed
静默发散检测（复现报告 §B.3）：epoch 10 后 loss>3 且持续 → 自动标记为 dead，取消后续
预测任务 overfit（新增）：对单张图反复预测，NCA predictor 应在 10 个 epoch 内将 L2 loss 压到 < 0.01。压不下 → NCA 连纯预测都学不动，检查 predictor 架构

附录 B：与审稿人预对峙
| 审稿人可能的攻击 | 预判回应 |
|---|---|
| "只是把 predictor 换了，novelty 不够" | 换 predictor 不是贡献——贡献是 (1) 首次表征并修复迭代预测器在 JEPA 中的稳定性问题，(2) 展示了 ViT predictor 不具备的三项能力。这些能力来自 NCA 的架构本质，不是调参能获得的 |
| "下游分数没超过 ViT predictor" | 本工作的目标不是超越 ViT，而是展示在不显著降低性能的前提下，NCA predictor 额外提供不确定性、可解释性和随时推理。这是 capability paper，不是 SOTA paper |
| "不确定性不就是 MC dropout 吗" | 有相似性但本质不同：MC dropout 是外挂的（训练后加 dropout），NCA 的随机点火是训练和推理原生一体的。我们会在实验里对标 MC dropout baseline |
| "稳定性方案不就是 spectral norm + 确定性模式吗，太简单" | 简单的方案能解决一个此前被忽视的系统性问题，这恰恰是好论文的特征。我们提供了从动力系统角度的理论分析，解释为什么这三件套是必要且充分的 |
| "为什么不用 NCA 替代 encoder 而只是 predictor" | NCA 的迭代本质使其不适合做 encoder（延迟高、不适合大批量预训练）。但在 predictor 位置，其局部等变性和随机性恰好转化为三项独有能力。encoder 的 ViT 贡献表示质量，predictor 的 NCA 贡献新能力——两者互补 |
