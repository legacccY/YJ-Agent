## 三、模型架构核心细节

### 3.1 Med-NCA (2D) 架构

```
输入: x ∈ R^(C×H×W)，其中 C=16 个通道（第0通道=原图，其余初始化为0）

粗层级 (Level 1):
  - 将输入下采样 d 倍（如 d=4，256→64）
  - NCA_coarse 迭代 s=64 步
  - 每步: 2× Conv2d(k=3) → concat identity → 1×1 Conv → BN → ReLU → 1×1 Conv
  - 残差更新: x_{t+1} = x_t + Δx_t * fire_mask

细层级 (Level 2):
  - 将粗层级输出上采样回原尺寸
  - 替换第0通道为原始高分辨率图像
  - 随机裁剪 patch（与粗层级同尺寸）
  - NCA_fine 迭代 s=64 步
  - 仅对 patch 计算 Dice Focal Loss
```

### 3.2 M3D-NCA (3D) 架构

```
- 使用 1× 3D Conv(k=7) 替代 2× 2D Conv
- 支持 n-level 架构（对比 Med-NCA 的 2-level）
- 每增加一层进一步降低 VRAM
- 引入 batch duplication 稳定训练
- 步骤数公式: s = max(W,H,D) / ((k-1)/2)
  - 例: k=7 → 感知域每步扩展 3 像素
```

### 3.3 关键超参数

| 参数 | 2D (Med-NCA) | 3D (M3D-NCA) |
|------|-------------|-------------|
| 通道数 C | 16 | 16 |
| 卷积核 k | 3 | 7 |
| 步数 s | 64 | 20-40 |
| 层级数 n | 2 | 1-2 (Hippocampus=1, Prostate=2) |
| Fire rate | 0.5 | 0.5 |
| 下采样因子 d | 4 | 视数据而定 |
| Batch size | - | 含 batch duplication |
| Optimizer | Adam | Adam |
| Loss | Dice Focal Loss | Dice Focal Loss |
| 参数总量 | <10^5 (<100K) | <10^5 (<100K) |

   - 下载 CheXMask 提供的肺部分割 mask
   - 预处理: resize 256×256, 70/30 split
   - 训练 Med-NCA 2D
   - 目标 Dice: ~0.941

2. X-ray Heart (PadChest + CheXMask)
   - 下载 PadChest 图像（需申请授权）
   - 使用 CheXMask 的心脏分割 mask
   - 预处理: resize 256×256
   - 目标 Dice: ~0.947

3. Dermatology (ISIC 2018)
   - 从 ISIC Archive 下载 Task 1 数据
   - 预处理: resize 256×256
   - ⚠️ 这是 RGB 3 通道图像，需修改输入通道
   - 目标 Dice: ~0.772

4. Breast Ultrasound (BUSI)
   - 从 Kaggle 或 Scholar 页面下载
   - 注意: 需将分类标签转为分割 mask
   - 目标 Dice: ~0.635
   - ⚠️ 论文中此数据集性能波动大（std=0.308），需多次实验
```

### 第 4 阶段：高级特性实现（2-3 周）

```
1. Pseudo Ensemble
   - 对同一输入运行 10 次推理（利用 fire rate 的随机性）
   - 取平均 mask
   - 验证 Dice 提升

2. NQM Score（NCA Quality Metric）
   - 利用 10 次推理的方差估计分割质量
   - 公式: NQM = Σ std(s) / Σ μ(m)
   - 用于 OOD 检测

3. Batch Duplication（仅 3D）
   - 在 batch 内复制同一图像（不同随机 patch）
   - 提升 n≥3 层级时的训练稳定性

4. NCA-VIS 可视化工具
   - 可视化 NCA 中间状态（16 通道）
   - 支持逐帧播放推理过程
   - 支持人为添加 artifact 测试鲁棒性
```

### 第 5 阶段：消融实验与改进探索（4-6 周）

```
1. 步数敏感性分析
   - 参考 Table 2，在 Breast 和 Liver 上实验不同步数
   - s ∈ {16, 32, 64} for Breast
   - s ∈ {10, ...} for Liver
   - 给出 perceptive range 的计算公式验证

2. 通道数/层级数消融
   - 改变通道数 C ∈ {8, 16, 32}
   - 改变层级数 n ∈ {1, 2, 3}
   - 记录 VRAM 使用变化

3. 鲁棒性测试
   - 图像缩放、平移、旋转不变性
   - MRI 采集 artifact (运动伪影、偏置场)
   - 噪声扰动

4. 与 baseline 详细对比
   - UNet（原论文已给）
   - nnUNet（2D 和 3D）
   - TransUNet
   - 还可以加入: SegFormer, SwinUNETR 等
```

```

## 七、从复现到创新的可能方向

完成复现后，以下是几个可探索的创新方向（供顶会投稿参考）：

1. **动态步数机制**：根据图像复杂度自适应决定迭代步数，减少冗余计算
2. **多头 NCA**：不同解剖结构用不同的 head，共享 backbone NCA
3. **NCA + Foundation Model**：用 SAM/MedSAM 的特征初始化 NCA 状态
4. **NCA for 半监督/弱监督**：利用 NCA 的 emergent behavior 实现少标注分割
5. **NCA for 视频/4D 医学图像**：利用时间维度的连续性
6. **硬件适配优化**：量化 + 剪枝，实测 Raspberry Pi 上的推理延迟
7. **改进 NQM**：将 NQM 用于 active learning 或 human-in-the-loop
