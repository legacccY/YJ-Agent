## 矩阵概览（18 run，~5 GPU·h，HPC 4 卡并行墙钟 ~1.5h）

### Block S — 数据地基（CPU，前置硬闸门）
- **S-0** Kaggle 4-class 溯源：OASIS 真实衍生 vs AI 合成 → 合成则停报改 OASIS。
- **S-1** 下载 6400 JPG + 4 类点数核对 + 登记 datasets.json。
- **S-2** 患者级 split：查有无 subject ID；无 ID → 退 OASIS 或仅当烟测。

### Block C — 分类器训练（GPU，L2 载体 + Gate1 acc 对齐文献）
- C-R-s1/s2/s3 = ResNet ×3 seed；C-V-s1/s2/s3 = ViT ×3 seed。
- 控制：同 split / 同分辨率 / 同 aug / ImageNet 预训练 / 类不均衡处理。
- **Gate1 判 acc 看 seed1 即可**；3 seed 为 L2 铺方差。超参全 `⟨TODO-R⟩` 待 researcher。
- 不均衡集 **必须同时看 macro-F1**（光看 acc 被多数类骗）。

### Block W — 子带置零 + L1 初验（GPU 推理，权重冻结）
- pytorch_wavelets `DWTForward(J=1)`→改子带→`DWTInverse` 重建回图像域喂模型。
- **W-base**（不置零，重建对照，验往返无损=置零生效锚点）+ W-LL/LH/HL/HH。
- 预期：LL 掉最多（结构），HH 掉最少（纹理/噪声）。
- ⚠️ **Gate1 只验「置零生效 + 趋势」，不下 L1 显著性结论（=Gate2 判据，防 HARKing）**。

### Block Q — Quantus faithfulness 非 nan 烟测（GPU 推理）
- Grad-CAM × {PixelFlipping, ROAD, IROF, insertion/deletion}，200 张子样本先验非 nan。
- insertion/deletion 若 Quantus 无独立实现 → PixelFlipping 等价 + 自实现 <50 行标清。
- **Gate1 只需 ≥1 组合算出非 nan**；SHAP/IG 全表留 Gate2/3。
