### 1.2 ★ 报告最有价值产出 = NCA 训练脆弱性（三档证据递进）

1. **fast-subclass 改 RNG → 单点对照**：把 fire-mask 的 `torch.rand` 从 CPU 挪 GPU，数学等价但改 RNG 流，prostate 临界配置下 0.0 全背景发散 vs 官方原版 0.672 收敛。→ 坐实「零偏离」红线。
2. **同 config 同 seed 两次 1000ep 均崩**（job 1436075 ep185 / 1436470 ep99）。
3. **n=9 seed 扫描（最硬，主推）**：0/9 存活至 ep1000；收敛/发散在 **ep1 即由不可控 GPU fire-mask 随机性掷定**（同 seed 42 两次 run ep1 loss 1.25 收敛 vs 4.33 发散）；好盆地也无安全期（seed 43 ep61、seed 46 ep122 健康数十~百 epoch 后单步断崖崩）；崩塌点 ep1~ep185 全程散布。
   - **机理**：NCA 随机 fire-mask（`cell_fire_rate=0.5` × 64 步）用 GPU rand 流 + CUDA 非确定 kernel，CPU seed 锁不住。

### 1.3 配套行为刻画（R1 官方 ckpt 重算）

- **C1 推理步数扫描（增值发现）**：官方训练 steps16，推理 steps=16 达峰 0.865，偏离即退化（steps32→0.808 / steps48→0.236 / steps64→**0.123 崩溃**）→ **Med-NCA 非 over-step-stable，推理步数必须匹配训练步数**。
- **V1 鲁棒性**：noise std0.4 最毒（0.865→0.046）；ghosting 最钝（−0.056）。
- **V2 NQM**：n=78，R5 Spearman ρ=0.4737 p≈3e-6，「内建质控」声明复现。
- **S1 确定性**：同 seed 复跑 max_abs_diff 0.0。

## 2. 创新线 — NCA-JEPA

**目录**：`project/meeting/Med-NCA/NCA-JEPA/`
**registry**：`nca-jepa` status=shelved（2026-06-17 用户拍板弃 NCA 主线）
**核心 idea**：JEPA 自监督世界模型里负责猜被遮挡 patch 的「预测器」，主流是 Transformer，本项目换成 NCA（迭代生长出答案）。NCA × JEPA × continual SSL 三者交叉 2026-06 全网零先例（真空白坐实）。

### 2.1 三卖点终局

| 卖点 | 判定 | 关键实测 |
|---|---|---|
| ① anytime 中途叫停 | ❌ 废 | A0+ early-exit ViT Q(1)=0.975 已极高（ViT 早退几乎无损）；NCA 各臂 Q(1)=0.63–0.85 < A0+ |
| ② 稳定性 | ✅ 真但扎实非惊艳 | SN 把 L_f 1.366→1.007 压 26%（3 seed std 0.0013）；但核心恒等式 `λ(I+g)=1+λ(g)` 是 i-ResNet 已知数学，非首创，且有人工反例 |
| ③ 抗遗忘 | ❌❌ 证伪且反转 | 等参对照（NCA 3.22M vs 小 ViT 3.28M）：NCA 遗忘比小 ViT **多 83%**（\|RF\| 0.2763 vs 0.1512），连 11.76M 大 ViT 都更抗遗忘 |

### 2.2 独家理论洞察（②的核心）

NCA 更新 `h_{t+1}=h_t+fire·δ`，Jacobian = `I+diag(fire)·J_δ`，那个单位阵 I 使谱半径**天然下界≈1**，SN 只压 J_δ 压不动 I → **严格收缩 L_f<1 结构性不可达**。谱半径下界系统验证（`eval/spectral_lower_bound.py`）：残差组 ρ≥1 跨 MLP/Attn/NCA/Conv 四族，SN 强度 c=0.3→2.0 扫 ρ=1.016→1.578 怎么压跨不过 1。
**诚实定位**：扎实（确无人系统做 SN 约束下残差 ρ→1⁺ 实证）但非惊天定律，精确 claim 须挂「SN 约束 + 非负反馈 g」条件。

### 2.3 反直觉负结果（③，可写诚实贡献）

「NCA 结构性抗遗忘」假说**不仅未成立、且方向反转** = NCA 结构本身更易遗忘。R1「只因容量小」被等参对照彻底排除。
**副发现**：NCA dLf≈0（续训前后谱半径几乎不动）= **「稳定 ≠ 抗遗忘」两者解耦**——动力学稳定性不蕴含表示抗遗忘。
**文献纠错**：立身 claim「NCAdapt 证 NCA 遗忘比 Transformer 小 100×」核实为假（NCAdapt arXiv:2410.23368 真实抗遗忘靠「冻骨干」工程非 NCA 内在，Table 4 自证不冻骨干 BWT −79%）。

## 3. 换路探路 — NCA-AB（Med-NCA 伞下）

**材料**：`Med-NCA/NCA-AB方向.md`（调研）+ `Med-NCA/NCA-AB理论.md`（数学分析）
**两方向**：A = 3D NCA 全分辨率体积分割（免 patch 省显存）；B = NCA 连续疾病轨迹生成（中间态=可解释渐进病变）。

### 3.1 方向 A → 判死（Gate 前）

- 卖点（免 patch / 多尺度信息传播）全被 **M3D-NCA 2023 + OctreeNCA 2025-08** 占（OctreeNCA 184MP 单遍分割少 90% VRAM）。
- 唯一缝隙「真免 patch（全分辨率单状态）」被 `理论.md` A.3 自己的显存账打死：512³ BPTT 17–68GB，4090 装不下，为甩 patch 反更费显存=自相矛盾。

### 3.2 方向 B → Gate3 轨迹证伪

四闸门：Gate0 ✅（新颖性留白：NCA 局部迭代生长=病变形态演化，仍空白）→ Gate1 ✅ → Gate2 ✅ → **Gate3 ❌**。

- **Gate1 persistence 能力发现（资产）**：NIH 胸片 64²，两 mode 收敛——damage（损伤恢复）best **26.5dB** ssim0.96 / reconstruct（糊图重建纹理）25.56dB。证明 NCA 在高纹理胸片上既能学成可修复 attractor 又能合成纹理（之前最大担忧被证伪）。
- **Gate2 纵向对数据核查（资产）**：NIH 30805 患者，≥2 次拍摄 13302（43.2%）；正常→异常转变对 14380 + 进展对 27722 = ~42k 疾病相关转变对可用。
- **Gate3 终判 FAIL**（job 1452901，Nodule 同 VP 400 对，64²）：NCA 学的是「停在 t0 附近的微漂移」非疾病进展（mono_to_t1 违反率 0.594 / change_in_roi_frac 0.49 < linear 0.72 / dist_from_t0@step64 仅 0.0145）；**且不是 NCA 特有失败，是 64²/~580 对下 Nodule 信号本就在分辨率/噪声底以下**——U-Net 一跳强基线 15.40 连「复制 t0」的 15.41 都没过。

## 7. 总结判决

NCA 在本组合台被从四个独立角度系统探过：**复现可信性、当 SSL 预测器、做 3D 分割、做疾病轨迹、update 稀疏度临界**。每条都跑到了可证伪的硬判据并诚实收口：

- **没有一条立成顶会旗舰**——天花板均在中等会议（TMLR/MIDL/MICCAI analysis track）。
- **但产出了一批扎实的负结果与诚实发现**：NCA 训练 RNG 脆弱性（fire-mask GPU-rand 锁不住 seed）、稳定≠抗遗忘解耦、谱半径结构性卡 ≥1、推理步数必须匹配训练步数、相变的确定性依赖梯度裁剪。
