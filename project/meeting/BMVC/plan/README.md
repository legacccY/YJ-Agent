## 文件清单

### 1. **plan2.md** - 图表生成完整指南

**内容**：7 张图的详细设计指南 + Python matplotlib 完整代码框架

**用途**：
- 需要画/修改图表时参考
- 每张图都有设计决策和代码框架
- 包含全局配色字典、matplotlib 设置、LaTeX 嵌入方式

**主要章节**：
- 全局配置（配色 + matplotlib 参数 + 输出格式）
- Fig 1-7 逐图指导（7 章节，每章含：数据来源、设计决策、需要替换的数据、常见错误、LaTeX 代码）
- Table 1-3 LaTeX 代码
- 投稿前终检清单

**适合场景**：
- "我要画 Fig 3，怎么做？" → 在本文件里搜 "Fig 3"
- "配色怎么定？" → 看"全局配置"章节
- "怎么嵌入 LaTeX？" → 每个图的最后有示例

## 🔄 使用工作流

### 需要理解整体策略
```
BMVC计划.md → 读前 5 章 → 明白为什么这个选题、大方向如何
```

### 需要制定日程
```
BMVC计划.md（第六章）→ 13 天执行计划 → 每天的 checklist
```

### 需要生成或修改图表
```
plan2.md → 搜索对应的图编号 → 找到"设计决策"和"你需要替换的数据" → 修改对应部分 → 重新运行 gen_bmvc_figures.py
```

### 需要跑实验
```
图表和实验计划.md → 找到对应的实验 → 复制代码框架 → 修改参数 → 运行 → 确保产出了正确的文件
```

### 投稿前最终检查
```
plan2.md 的最后 → 投稿前终检清单 → 逐项确认
```
